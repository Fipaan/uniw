About This Project

This project is a demonstration of HTML, CSS, Bootstrap, JavaScript and jQuery. It includes multiple pages, a responsive layout, an interactive carousel, gallery modal, form validation and jQuery capabilities.

Project Structure

  • assets: assets/_page_/_asset_name_ (e.g. assets/home/post1.png)
  • pages: _page_/index.html (e.g. home/index.html)
  • scripts per page: _page_/script.js (e.g. home/script.js)
  • styles per page: _page_/style.css (e.g. home/script.css)
  • blog/index.html (posts list + gallery modal)
  • post/index.html (single post + comments)
  • about/index.html (this page with information about project)
  • contact/index.html (contact form)
  • common part: common.css, common.js
